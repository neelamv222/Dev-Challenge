Table Description:

This is a description of what I have done. I have created a table with rows for each currency pair, which includes columns as:

  • Name
  • Current Best Ask Price
  • Current Best Bid Price
  • Best Ask Price Last Changed
  • Best Bid Last Changed
  • Open Ask
  • Open Bid

The table is getting sorted (and remain sorted) depending on the value of column "best bid last changed". I have displayed it in descending order. Depending on the value of column "best bid last changed", rest of the columns are getting sorted accordingly. Hightest value (of "best bid last changed" column) is shown first.
